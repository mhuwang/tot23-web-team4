# 中国国家博物馆云平台项目网页端
本项目参考与 Panjiachen 的后端模板 vue-admin-template，因此有问题可以参考原项目，链接附后

## 安装

```bash
# 拉取代码
git clone https://github.com/RexJoush/MuseumByCloud.git

# 进入目录
cd MuseumByCloud

# 安装依赖
npm install

# 启动项目
npm run dev
```

会自动打开项目 http://localhost:9528

## 编译打包

```bash
# 测试环境
npm run build:stage

# 生产环境
npm run build:prod
```
